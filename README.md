# cuda-vector-types
Testing CUDA Built-In vector types. And how they are translated to PTX and SASS.

From: https://docs.nvidia.com/cuda/cuda-c-programming-guide/#built-in-vector-types

### Built-in Vector Types (interesting ones):
- Type    Alignment
- float1	4
- float2	8
- float3	4
- float4	16
- double1	8
- double2	16
